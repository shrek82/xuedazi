# 游戏成功

## 各种模式朗诵逻辑

### 单字模式

例如:大，拼音:da
进入后完整朗读整句“大”，然后用户输入刚完第一个字的完整的拼音“da”后，朗读“大”；因为是单字，所以跳过完整阅读整句“大”，最多只完整的读2次，避免3次

### 词语模式

例如:太阳，拼音:tai yang
进入后完整朗读整句“太阳”
用户输入完第一个字的完整的拼音“tai”后，朗读“太”；
用户输入完第二个字的完整的拼音“yang”后，朗读“阳”；
完整的朗读“太阳”
确保全部拼音输入后，能听到一次完整的“太阳”

### 成语模式
例如：马到成功
进入后完整朗读整句“马到成功”，拼音:ma dao cheng gong
用户输入完第一个字的完整的拼音“ma”后，朗读“马”；
用户输入完第二个字的完整的拼音“dao”后，朗读“到”；
用户输入完第三个字的完整的拼音“cheng”后，朗读“成”；
用户输入完第四个字的完整的拼音“gong”后，朗读“功”；
完整的朗读“马到成功”

## 短句，短文，古诗，文章模式
同 成语模式


要求总结：
1. 单字模式，词语模式，成语模式，短句，短文，古诗，文章模式，都要完整朗读整句，然后根据用户输入的拼音，一个一个字的朗读。
2. 确保每个字都能听清楚，全部拼音输入后，还能听到最多一次完整的朗读。
3. 字，词语，成语，短句，短文，古诗，文章模式等模式，为了防止积压朗读，可以采用智能朗读语速；
4. 旁白与首句朗读并存时，若用户尚未输入，则旁白结束后朗读首句；若用户已开始输入，立刻停止旁白并立即朗读首句，避免输入完成后才听到首句。
