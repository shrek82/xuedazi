很有创意的想法！这种"有性格的旁白"能让游戏更有趣。以下是设计方案：
🎭 旁白类型设计
1. 鼓励类（输入正确时）
| 场景 | 旁白文案示例 |
|------|-------------|
| 普通正确 | "继续！"、"漂亮！"、"对！" |
| 连击中 | "手感不错嘛~"、"停不下来？"、"小意思！" |
| 长词完成 | "厉害了！"、"这都能打对？"、"学长了不起！" |
| 快速输入 | "手速可以啊"、"神速！"、"键盘要冒烟了" |
2. 表扬类（完成阶段时）
| 场景 | 旁白文案示例 |
|------|-------------|
| 过关 | "不愧是你！"、"这一关难不倒你嘛" |
| 高分 | "新纪录！给自己鼓个掌" |
| Boss 击败 | "太强了！Boss 都怕你了" |
| 满血过关 | "完美！一字不错！" |
3. 轻微讥讽类（错误时）
| 场景 | 旁白文案示例 |
|------|-------------|
| 普通错误 | "哎呀~"、"手滑了？"、"认真点嘛" |
| 连续错误 | "键盘烫手？"、"今天没吃饭？"、"别急别急" |
| 低级错误 | "这个也错？"、"眼睛瞪大点" |
| 连击被打断 | "可惜了~"、"差一点点！" |
4. 冷幽默类（特殊场景）
| 场景 | 旁白文案示例 |
|------|-------------|
| 长时间不输入 | "睡着了？"、"键盘离家出走了" |
| 反复卡同一词 | "这个词跟你有仇？"、"再试亿次？" |
| 差一点满血 | "差一个！气不气？" |
| 打字速度忽快忽慢 | "精神分裂？" |
---
🎙️ 旁白角色风格
| 角色 | 风格描述 | 示例 |
|------|----------|------|
| 傲娇猫 | 表面嫌弃，实际关心 | "哼，还行吧...才不是在夸你" |
| 毒舌兔 | 犀利吐槽，但无恶意 | "这都能错，你礼貌吗？" |
| 老爷爷 | 和蔼可亲，循循善诱 | "慢慢来，不着急，爷爷等你" |
| 机器人 | 冷静客观，一本正经 | "检测到错误输入，请人类重新校准" |
